How this works: Each listing is assigned a Census place name via point-in-polygon lookup against the free Census TIGER/Line shapefile for South Carolina. Names like "James Island CDP", "Goose Creek city", and "Summerville town" are official Census-defined boundaries — stored once in the DB as place_name, never needing re-lookup. Toggle places on the left to see filtering in action.
Implementation Plan
STEP 01 — ONE TIME SETUP
Download shapefile
Download tl_2024_45_place.zip from Census TIGER (~3 MB). Unzip to data/ in your project. Install geopandas + shapely.
STEP 02 — PIPELINE
Enrich at ingest
Pipeline calls get_place_name(lat, lon) for each new listing. Stores result in place_name column. Costs zero API calls — pure local geometry.
STEP 03 — FILTERING
Query by place
Dashboard excludes WHERE place_name IN (avoided_list). Preferences page stores avoided place names. Autocomplete from shapefile's name list.
